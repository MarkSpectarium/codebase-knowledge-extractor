Send a message to another WezTerm pane and submit it. Usage: /send-to-pane <pane-id> <message>

Run: send-to-pane $ARGUMENTS

> **Note**: The `send-to-pane` function is available via BASH_ENV when spawned by spawn-agent.js. If not available, use: `node $AGENT_SETUP_PATH/prompts/scripts/send-to-pane.js $ARGUMENTS`
